SHA-256 Cryptographic Hashing

Every certificate is hashed using NIST-standard SHA-256. The hash is stored permanently on Ethereum — a single character change produces a completely different hash, making tampering instantly detectable.

🦊

MetaMask Wallet Signing

Certificates are issued by signing a real Ethereum transaction with your MetaMask wallet. This proves the issuer's identity cryptographically — no passwords, no central authority.

⛓️

Ethereum Smart Contract

A deployed Solidity smart contract holds all certificate records on-chain. Once written, data cannot be deleted or altered — guaranteed by Ethereum's immutable ledger.

📁

Certificate File Hashing

Upload the actual certificate PDF or image. Its SHA-256 fingerprint is computed in-browser and stored on-chain. Later, upload the same file to verify it hasn't been altered since issuance.

Etherscan Verified

Every issuance transaction is viewable on Sepolia Etherscan. Anyone in the world can independently verify a certificate without trusting this website.

♾️

Immutable Records

Certificates on the blockchain are permanent. No delete, no update — once issued and confirmed, a certificate exists forever in Ethereum's history, backed by a global network of validators.
